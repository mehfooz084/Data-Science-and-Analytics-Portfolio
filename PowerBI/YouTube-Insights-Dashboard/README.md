# YouTube Insights Dashboard

## Overview
This Power BI dashboard analyzes YouTube channel performance through views, subscribers, likes, comments, categories, and advertising revenue trends.

## Tools Used
- Power BI
- DAX
- Data Visualization

## Key Insights
- Total Views reached **499B** across analyzed channels.
- Total Likes exceeded **14B**, indicating strong audience engagement.
- Total Comments crossed **2B**, reflecting high viewer interaction.
- Ad Revenue grew steadily from 2019 to 2023, peaking at **32K** before declining in 2024.
- **Music** recorded the highest average views among content categories.
- **T-Series** led subscriber counts among the top analyzed channels.
- Top-performing channels consistently generated significantly higher average views than other creators.
