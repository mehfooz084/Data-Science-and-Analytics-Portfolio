# AI Impact on Jobs Dashboard

## Overview
This Power BI dashboard analyzes the potential impact of AI on industries and job roles through AI adoption, automation risk, salary impact, and skill demand growth.

## Tools Used
- Power BI
- DAX

## Key Insights
- Average AI Adoption: 45.21%
- Average Automation Risk: 41.81%
- Customer Support and Truck Drivers showed the highest automation risk.
- Data Analysts and Software Engineers showed the lowest automation risk.
- Truck Drivers recorded the highest salary decline (-30.25%).
- Software Engineers and Data Analysts showed the strongest skill demand growth.
